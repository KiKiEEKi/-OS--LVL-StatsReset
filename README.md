Описание:
Сбрасывает статистику в конце каждого месяца.
Плагин учитывает сколько дней в каком месяце.

Проверка даты и время идет при запуске плагина.
Лучшей вариант использование это задать время перезагрузки сервера каждый день.
Что полезно в целом это делать.

Требования:
Metamod
Sourcemod
Levels Ranks

Переменные:
Конфиг создаться автоматически в
cstrike/cfg/sourcemod/[OS][LVL]StatsReset.cfg

Установка:
Закинуть папки из архива в папку sourcemod
